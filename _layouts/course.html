Course Overview

This course introduces the theory, algorithms, and applications of distributed optimization for machine learning and AI. Topics include large-scale optimization, federated learning, consensus algorithms, and resource-aware designs for modern distributed systems.

  • Prerequisites: Linear Algebra, Probability, and Basic ML background
  • Format: Lectures + Homework + Project
